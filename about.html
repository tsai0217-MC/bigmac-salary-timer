About Big Mac Salary Timer

A real-time salary visualization tool inspired by the Big Mac Index.

What This Website Does

Big Mac Salary Timer is an interactive web application that helps users visualize how quickly their salary accumulates in real time. The website also compares earnings with the price of a Big Mac across different countries using Big Mac Index data.

Instead of only showing salary as a number, this project turns income into a more intuitive and visual experience. Users can see how much they earn over time, how many Big Macs their income can buy, and how purchasing power differs across countries.

About the Creator

Hi, I'm Meng-Chen Tsai, a UC Berkeley Master of Biotechnology graduate and independent developer. My background combines biotechnology, scientific research, data analysis, and software development.

Although my academic training is in biotechnology, I am also passionate about programming and building practical web applications. I enjoy creating tools that make data, economics, and scientific concepts easier to understand through interactive design.

Project Purpose

This project was built as an independent personal project to explore the intersection of salary visualization, purchasing power, economic data, and front-end web development.

Big Mac Salary Timer is not affiliated with McDonald's Corporation, The Economist, or any related organizations. It is intended for educational and informational purposes only.

Future Goals

Future updates may include additional economic indicators, global purchasing power comparisons, interactive maps, and more educational tools related to economics and personal finance.
